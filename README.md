# 品牌商關鍵字括號配對檢查工具

一個純前端的小工具，用來檢查關鍵字清單中的括號 `(` `)` 是否配對正確。

## 功能

- 逐行檢查括號配對是否正確
- 標示錯誤類型（多了 `)` 或少了 `)`）
- 統計總行數、正常行數、錯誤行數
- 可篩選只顯示錯誤行
- 一鍵複製所有錯誤行內容

## 使用方式

1. 開啟線上網址：[https://hankyang-sketch.github.io/brand-partnership-tools/](https://hankyang-sketch.github.io/brand-partnership-tools/)
2. 將關鍵字清單貼入輸入框（每行一組）
3. 點擊「開始檢查」
4. 檢視結果，錯誤行會標示紅色

## 說明

本工具純前端運算，所有資料僅在瀏覽器本機處理，不會上傳或傳送至任何伺服器。
